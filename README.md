# wdd130-Rose
The final project for the WDD130 class, a website designed for a client.

The content of the page (text and images) were produced by the client, using AI generation. However, none of the code (html, css, and javascript) was produced this way. https://www.w3schools.com/ was used as a reference to assit with programming. Brush Script MT was obtained from https://freefonts.co/fonts/brush-script-mt-regular. The page's favicon was generated using https://favicon.io/.
