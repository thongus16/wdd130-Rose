# wdd130-Rose
The final project for the WDD130 class, a website designed for a client.

The content of the page (text and images) were AI generated by the client, but none of the code (html, css, and javascript) is. https://www.w3schools.com/ was used as a reference to assit with programming. Brush Script MT (the font used for headers), was obtained from https://freefonts.co/fonts/brush-script-mt-regular. https://favicon.io/ was used to generate the page's favicon.
